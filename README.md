# HydrAID-Neurosymbolic-AI-for-Predictive-Biomarkers-in-Cancer-Immunotherapy
HydrAID combines a Knowledge Graph (RDF/SPARQL + SHACL) and neuro-symbolic RAG to discover predictive biomarkers in immunotherapy (NSCLC) with explainability and traceability. Includes BEO, YARRRML→RMLMapper, and an NL→SPARQL agent.
